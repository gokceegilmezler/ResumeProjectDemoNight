## 📌 Proje Özellikleri
- 🔐 Kullanıcı kayıt ve giriş sistemi (ASP.NET Identity)
- 📬 Mesajlaşma sistemi (Inbox / Send Message)
- 🧩 ComponentPartial yapısı ile modüler mimari
- 📊 Entity Framework Core ile veritabanı işlemleri (CRUD)
- 🎨 Bootstrap ile responsive (duyarlı) tasarım
- 📁 Dinamik içerik yönetimi (Deneyimler, Hizmetler, Projeler vb.)


## 🛠️ Kullanılan Teknolojiler
- Backend: ASP.NET Core MVC, C#
- Frontend: Razor View Engine, HTML, CSS, Bootstrap
- Database: SQL Server
- ORM: Entity Framework Core
- Mimari: Component-Based yapı (Partial & ViewComponent)


## 📚 Öğrendiklerim

Bu proje ile birlikte:

ASP.NET Core MVC yapısını uçtan uca uygulamayı
Katmanlı mimariyi ve modüler geliştirmeyi
Veritabanı ile etkileşim kurmayı
Gerçek bir proje geliştirme sürecini deneyimleme fırsatı buldum.


## 📎 Not

Bu proje, yazılım geliştirme sürecinde kendimi geliştirmek amacıyla hazırlanmıştır ve geliştirilmeye açıktır.
